# 행동 강령 (Code of Conduct)

이 프로젝트는 개방적이고 존중하는 협업 환경을 지향하며,
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1 을 채택합니다.

## 요약
- 배려 있고 존중하는 언어를 사용합니다.
- 다른 관점·경험을 존중합니다.
- 건설적인 비판을 주고받습니다.
- 괴롭힘·차별·인신공격은 허용되지 않습니다.
